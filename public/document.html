快速入门指南

注册账号

首次使用API

Python
import requests

API_KEY = "your_api_key"
url = "https://api.voiceconversion.com/v1/convert"

payload = {
    "text": "欢迎使用语音转换API",
    "voice_model": "female_1"
}
headers = {
    "Authorization": f"Bearer {API_KEY}",
    "Content-Type": "application/json"
}

response = requests.post(url, json=payload)
print(response.json())

API介绍

语音转换API提供文本转语音和语音转换功能，支持多种语音模型和自定义参数。API采用RESTful设计，所有请求均通过HTTPS进行。

基础URL

基础URL
https://api.voiceconversion.com/v1

支持的HTTP方法

  • GET: 获取资源信息
  • POST: 创建资源或执行操作
  • PUT: 更新资源
  • DELETE: 删除资源

认证方式

API密钥认证

所有API请求需要在HTTP头部包含认证信息：

HTTP头部
Authorization: Bearer {your_api_key}

密钥管理

  • 定期轮换API密钥（建议每90天）
  • 不要在客户端代码中硬编码API密钥
  • 使用环境变量或配置文件管理密钥

权限控制

API密钥可关联不同权限级别：

  • 管理员：完全访问权限
  • 开发者：API调用权限
  • 只读：仅能获取资源信息

请求

参数名 类型 必选 描述

错误代码

常见错误码

状态码 错误码 描述 解决方法

API调用优化

  • 监控异常API使用模式

性能优化

  • 控制文本长度，避免超长请求
  • 选择合适的音频格式（MP3比WAV体积小）
  • 预加载常用语音模型
  • 合理设置超时时间（建议5-10秒）

音频质量提示

输入文本优化

  • 避免使用过长的连续文本
  • 适当添加标点符号，改善停顿效果
  • 使用自然的口语化表达
  • 避免使用特殊符号和表情

音频参数设置

  • 语速建议设置在0.9-1.1之间，保证清晰度
  • 重要内容可适当提高音量（1.1-1.2倍）
  • 情感文本可调整音调增强表现力
  • 背景噪音环境下使用较高音量

输出格式选择

  • 网络传输：选择MP3格式（较小体积）
  • 高质量需求：选择WAV格式（无损音质）
  • 移动端播放：建议采样率44.1kHz，比特率128kbps
  • 语音识别场景：选择8kHz采样率，单声道

常见问题

账户相关

Q: 免费版和付费版有什么区别？
A: 免费版有请求次数和功能限制，付费版提供更高的速率限制、更多语音模型和优先技术支持。

Q: 如何升级账户？
A: 在控制台"账户设置"→"订阅计划"中选择合适的套餐，完成支付即可升级。

技术相关

Q: 支持哪些语言？
A: 目前主要支持中文（普通话），英文支持正在测试中，其他语言计划在未来版本推出。

Q: 音频文件能保存多久？
A: 免费账户保存7天，标准账户保存30天，企业账户可自定义保存时间。

Q: 是否支持离线语音转换？
A: 目前仅提供在线API服务，离线SDK需联系销售团队获取企业定制方案。

Web前端集成

JavaScript
// 使用Web SDK
const voiceConverter = new VoiceConversionSDK({
  apiKey: 'your_api_key'
});

// 文本转语音
voiceConverter.textToSpeech({
